## 개요 

-	#### Git이란?

​	**분산버전관리시스템**으로 코드의 버전을 관리하는 도구이다. 2005년 리눅스 커널을 위한 도구로 리누스 토르발스가 개발 했으며, 컴퓨터 파일의 변경사항을 추적하고 여러 명의 사용자들 간에 해당 파일들의 작업을 조율하는 역할을 한다.
